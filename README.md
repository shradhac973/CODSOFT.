# CODSOFT Data Science Internship Projects

This repository contains the projects completed during my Data Science Internship at CODSOFT. These projects demonstrate the application of data analysis, machine learning, and predictive modeling techniques using Python.

## Projects Included

### 1. Titanic Survival Prediction

Built a machine learning model to predict whether a passenger survived the Titanic disaster based on features such as age, gender, passenger class, and fare.

**Technologies Used:**

* Python
* Pandas
* NumPy
* Scikit-learn

---

### 2. Sales Prediction Using Python

Developed a predictive model to estimate product sales based on advertising expenditures across different marketing channels.

**Technologies Used:**

* Python
* Pandas
* Matplotlib
* Scikit-learn

---

### 3. Credit Card Fraud Detection

Implemented a machine learning model to identify fraudulent credit card transactions using transaction data and classification techniques.

**Technologies Used:**

* Python
* Pandas
* Scikit-learn

---

### 4. Movie Rating Prediction

Built a regression model to predict movie ratings using features such as genre, director, actors, duration, votes, and release year.

**Technologies Used:**

* Python
* Pandas
* Scikit-learn
* Random Forest Regressor
